========================================
    AI 시험지 생성기 - 완전 자동 설치
========================================

📋 개요
- 수학 시험지를 생성하고 편집할 수 있는 웹 애플리케이션
- LaTeX 수식 지원, PDF/Markdown 내보내기 기능
- Node.js 자동 설치로 별도 준비 없이 바로 사용 가능

🚀 사용법 (매우 간단!)
1. "start-simple.bat" 파일을 더블클릭
2. 자동 설치 과정을 기다림 (최초 실행시만)
3. 브라우저에서 자동으로 열리는 웹페이지 사용

📦 자동 설치 기능
- Node.js 미설치시 자동 다운로드 및 설치
- 프로젝트 의존성 자동 설치
- 빌드 파일 자동 생성
- 서버 자동 실행

⚠️ 주의사항
- 인터넷 연결 필요 (최초 실행시)
- Windows 관리자 권한 필요할 수 있음
- 방화벽에서 Node.js 허용 필요할 수 있음

🔧 수동 설치 (자동 설치 실패시)
1. https://nodejs.org/ 에서 Node.js LTS 버전 다운로드
2. 설치 후 컴퓨터 재시작
3. "start-simple.bat" 재실행

💡 사용 팁
- Ctrl+C로 서버 종료
- 브라우저에서 http://localhost:3000 수동 접속 가능
- PDF 저장: 브라우저 인쇄 기능 사용 (Ctrl+P)
- 파일 저장: JSON 형식으로 저장/불러오기 가능

🎯 주요 기능
✅ 수학 문제 작성 및 편집 (LaTeX 수식 지원)
✅ 실시간 미리보기
✅ 문제와 해설 분리 출력
✅ PDF로 내보내기
✅ Markdown 파일로 내보내기  
✅ JSON 파일로 저장/불러오기
✅ 폰트 크기, 줄 간격, 문제 간격 조정
✅ 반응형 웹 디자인

📞 문제 해결
- 설치 실패: 관리자 권한으로 실행
- 브라우저 미실행: 수동으로 http://localhost:3000 접속
- 포트 충돌: 다른 프로그램 종료 후 재실행
