# 시험지 생성기

이 앱을 로컬에서 실행하는 데 필요한 모든 것이 포함되어 있습니다.

## 로컬 실행

**사전 요구사항:** Node.js

1. 종속성 설치:
   `npm install`
2. 앱 실행:
   `npm run dev`
